Basalt - Blockchain App

Basalt - Blockchain Appclose

For my final year project I wanted to get experience with new technologies I hadn't used before. I created a Blockchain application to allow for the storage and sharing of academic credential documents across as simulated Ethereum network.

The front end was developed using React JS and Firebase provided the authentication mechanism. The Truffle suite was used to create and test my smart contracts (code that runs on a Blockchain).

Technologies: React JS, Truffle, IPFS, Firebase, Web3.js, Solidity
Hoot - Alzheimer Patient Device

View the repository

Hoot - Alzheimer Patient Deviceclose

I was part of a 4 person team tasked with creating a prototype device to help aids carers of Alzheimer's patients.

The device was created using a Raspberry Pi and entailed a series of infrared senders & recievers to check if a patient has moved between rooms. Reports are sent from the device to the carers email and can also be viewed on the accompanying web app.

Technologies: Python, Flask, Raspbery Pi
Pneumonia X-ray Detecter

View the repository

Pneumonia X-ray Detecterclose

I was part of a 3 person team who created an application to aid doctors in the detection of pneumonia in patients.

The model was trained using TensorFlow using X-ray images from a dataset we found on Kaggle. The application was presented as a Flask web app.

To use this applicaion the doctor submits an X-ray which is then sent to the trained model. The image is analysed and the verdict is returned along with a confidence percentage.

Technologies: Python, Flask, TensorFlow, SKLearn
